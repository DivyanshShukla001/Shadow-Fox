# Divyansh Shukla - Portfolio Website

## Overview

This is my personal portfolio website showcasing my skills, projects, education, and experience as a Full-Stack Web Developer and B.Tech CSE student at KIIT University.

## Features

- Responsive Design
- Modern User Interface
- About Me Section
- Skills Showcase
- Experience Timeline
- Project Portfolio
- Education Details
- Contact Information
- Smooth Navigation

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts

```
Shadow-Fox/
│
```

## Getting Started

1. Clone the repository

```bash
git clone https://github.com/DivyanshShukla001/Shadow-Fox.git
```

2. Open the project folder

```bash
cd Shadow-Fox
```

3. Run the project

Open `index.html` in your browser.
